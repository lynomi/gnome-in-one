# Gnome in One

## https://lynomi.github.io/gnome-in-one/

## Functionality

- **Build** - Place blocks, clear blocks, adjust angle and power of the ball, preview simulation trajectory.
- **Swing** - Physics simulation runs. Player can reset to the build phase during the simulation.
- **Result** 
    - Win if the ball reaches the hole. Player is given the option to retry or advance to the next level.
    - Lose if the ball does not reach the hole. Player is given the opion to retry.
